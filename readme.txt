This plugin changes the unique theme way of thinking and propose a user-agent dependant switching theme. You'll be able to change the theme of your blog if your viewer has firefox, safari, an ipad, or android ; you may also want a SEO optimized theme for googlebot !

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `swf_agent` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Download some themes, add some rules to the plugin admin panel and watch

You may also find usefull this firefox plugin : "User Agent Switcher" (available in mozilla repo).
